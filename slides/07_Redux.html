Going towards Redux

Problems of immutable state

  • State can be huge & complex
  • Preserving immutability is fragile
  • Tracking changes is complicated
  • ...
Redux is doing this for us

Concept - Handling immutable state

  • Start point -> initial state
  • Protect/Encapsulate state -> store
  • Applying changes -> actions
  • Tracking changes -> listeners, subscribers
