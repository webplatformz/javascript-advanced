String API: Trimming

" abc ".trimLeft(); // "abc "
" abc ".trimRight(); // " abc"
" abc ".trim(); // "abc"

do it yourself again

" abc ".replace(/^\s+/g, ''); // "abc "
" abc ".replace(/\s+$/g, ''); // " abc"
" abc ".replace(/^\s+/g, '').replace(/\s+$/g, ''); // "abc"
